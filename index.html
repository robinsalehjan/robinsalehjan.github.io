Projects
  • Soundpick:

    Soundpick is a Raycast app extension that enhances your music listening experience on MacOS. It allows you to conveniently view and manage your music playback across different devices. You can see which device is currently playing music via Bluetooth, explore available devices for music playback, and effortlessly connect to or disconnect from these devices.

  • Trending Hashtag Graph:

    A interactive web app developed in Elixir and Cytoscape that visually displays the most recent trends on Twitter. It fetches the top 10 trending hashtags and accompanies each with the ten most recent tweets using that hashtag. These trends and tweets are presented as a series of interconnected graphs, with the most recent graph appearing first. This layout makes it easy to see the latest buzz on Twitter in a visually engaging and organized manner.

  • Supercharged:

    A collection of scripts that orchestrates the setup and configuration of a MacOS environment for development, streamlining the installation of a wide array of tools and applications like Git, Docker, and Visual Studio Code.
